DRIVER CLASS

The driver for my product allows the user to create a trust. The user gets to choose:

  • the name of the trust
  • the number of companies in the trust
  • the product of the trust
  • the price the companies will fix the product to

Companies for the trust are generated at random with random numbers for names. When the trust is generated, the driver sorts the members of the trust in ascending order by trust and prints them. Then the driver asks the user if he/she wants to progress 1 day so that a random number of people can "buy" the product. The driver then asks the user if he/she wants to change the price of the fixed product for a certain product. If the user says yes, they can choose the company that will change the price by its name. If the price is set to lower than the fixed price then the company is immediately kicked out of the trust, if it's too high then the price is set back down to the trust's price.
